# 색상 코드 추출 SHColorPicker
색상코드를 확인하는 가벼운 소프트웨어입니다.

마우스 커서를 기준으로 RGB 색상 코드(`255,255,255`) 또는 HTML 코드(예: `#ffffff`)를 볼 수 있습니다.

급하게 사용하기에 좋습니다.

![스크린샷](./screenshots/screenshot_1.gif)
